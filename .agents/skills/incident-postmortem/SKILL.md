## Required context

Before acting:

1. Read `AGENTS.md`.
2. Read `docs/ai-project-memory/00-project-index.md`.
3. Read the current branch memory file if one exists.
4. Inspect actual repository files before making conclusions.

## Workflow

1. Preserve timeline and evidence.
2. Identify impact, root cause, detection gap, and corrective actions.
3. Add regression/security tests.
4. Update security model, testing matrix, and release checklist.
5. Produce action items with owners.
