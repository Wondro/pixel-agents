description: Research a technical approach or vendor integration and produce options with tradeoffs.

## Required context

Before acting:

1. Read `AGENTS.md`.
2. Read `docs/ai-project-memory/00-project-index.md`.
3. Read the current branch memory file if one exists.
4. Inspect actual repository files before making conclusions.

## Workflow

1. Define the decision or unknown.
2. Have `research_analyst` verify current docs and constraints.
3. Have `staff_architect` compare options.
4. Include cost, risk, security, lock-in, and implementation complexity.
5. Record durable decisions as ADRs when accepted.
