## Required context

Before acting:

1. Read `AGENTS.md`.
2. Read `docs/ai-project-memory/00-project-index.md`.

4. Inspect actual repository files before making conclusions.

## Workflow

1. Have `ux_accessibility` inspect changed UI.
2. Check keyboard navigation, focus, labels, errors, mobile layout, empty/loading states, and plan/billing messaging.
3. Recommend or implement minimal fixes.
4. Add component/e2e tests where available.
